# SFSSPlatform Web UI

This React app uses Vite, TypeScript, pnpm, Tailwind CSS, TanStack Query, Vitest, and Testing Library.

UI direction:

- Dark theme by default.
- Product-grade developer-learning interface, not generic AI dashboard styling.
- Draw structural inspiration from leading developer learning platforms: direct catalog navigation, lesson/task focus, code-first exercise surfaces, progress clarity, and dense but readable information hierarchy.
- Avoid glassmorphism, decorative gradients, oversized cards, fake metrics, hero sections inside the app, and ornamental badges.
- Prefer restrained panels, clear tables/lists, file-tree/editor layouts where relevant, practical typography, and functional state indicators.

Run locally:

```bash
pnpm install
pnpm dev
```
